# 휴먼톡톡

### 프로젝트 세팅
- 프로젝트를 Clone하고, Vercel과의 연동을 진행주세요.
  - `now-cli`를 설치하고, `$ now login`으로 사용 중인 Vercel 계정으로 로그인해주세요.
  - 프로젝트 루트에서 `$ now`로 프로젝트를 연동해주세요.
- `$ now env pull`을 통해 개발용 환경변수 파일을 받아주세요.
- `$ yarn`을 통해 라이브러리를 설치해주세요.
- `$ yarn dev`, 끝입니다!
